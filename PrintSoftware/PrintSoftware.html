Introduction

RControl Printing Software Library can:

  • Load bitmap files (BMP, PNG, JPEG, GIF)
  • Load DXF files and generate internal pixel data according to printing conditions
  • Convert image data into print data and transmit it to the printhead driver
  • Providing a large printing memory buffer (upto 2 GB for each printhead)
  • Reducing printing time by eliminating transferring data each time but storing full page data in the memory buffer
