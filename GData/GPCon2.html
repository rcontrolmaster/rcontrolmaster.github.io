GData GPCon2

GPCon is a printing controller for two head drivers.

  • communication channels for RDriver
    pcCOM protocol (RControl proprietary protocol)
    robust RS422/485 signal levels.
  • Large printing image buffer (page memory) for ecah channel
  • Each channel can perform spitting and printing independently (asynchronously).
    Alternatively, both channels can operate synchronously using the bus trigger and encoder signals.
  • Easy implementation using a standard RJ45 connector.
    A standard Ethernet cable can be used.

IP address setting

How to get IP and Channel Information

IP address = ConRTL Base Address (DIP switch) + GPCon2 Address(Rotary DIP switch).
For example, given Base Address = 192.168.0.100, the IP address becomes 192.168.0.102 when the rotary DIP switch value of GPCon2 is 2.
