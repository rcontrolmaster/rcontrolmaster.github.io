GData GPCon2

Overview

● The GPCon2 is a printing controller for 2 printhead drivers.

● Gigabit Ethernet connectivity for high-speed data transfer.

Module Information

● Transfer rate : 1 Gbps

● Printing memory : 2 GBytes per printhead

● Page memory system can reduce the time of transferring image data drastically.
