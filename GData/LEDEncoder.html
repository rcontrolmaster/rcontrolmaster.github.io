GData EncoderLED-u

EncoderLED-u module is a all-in-one timing path controller.

  • Position Encoder
    • supports Incremental Encoder with RS422 line driver
    • standard DSUB 9 pin connector.
  • Trigger generator
    • BNC connector for isolated trigger out.
    • LED indicator showing the trigger output state.
  • LED strobo light source for dropwatcher.
    • 2 channel LED controller.
    • screw block connector for convenient wiring

Encoder controller

  • standard Encoder quadrature decoding.
  • Provides 5 V power for the encoder sensor.
  • Encoder emulator and counter for testing and experimentation.
  • Trigger generator for synchronization and emulation.
    The trigger signal is routed to the internal bus and to the BNC connector for external synchronization.
  • TrigOut
    Isolated trigger output signal.
    5 V TTL level, 1 µs pulse duration (programmable).
Signal Description
AP / AN A-phase RS422 signal
BP / BN B-phase RS422 signal
+5V, GNDS 5 V output, 400 mA
DP / DN Not used

The +5V connection may be left unconnected if not required.

Encoder emulator

By providing emulated encoder signals, printing simulation can be performed very easily.

Trigger Generator

  • frequency : 1 Hz ~ 100 kHz
  • count : 67,108,863 max
  • trigger out mux : trigger generator output, position encoder 4x signal, position encoder signal R, position encoder signal A
  • electrically isolated trigger out signal.

LED strobo light controller

  • Independent dual-channel output.
  • Supports driving 7 W LEDs.
  • Delay time and pulse duration can be configured.
    Clock resolution : 10 ns
    Duration time : > 100 ns
    Supports one-shot and two-shot modes.
  • Synchronized by the trigger signal (trigOut).
  • Terminal block connector : 5.08 mm pitch
    OQ0454510000G

    a0 : anode channel 0, c0 : cathode channel 0
    a1 : anode channel 1, c1 : cathode channel 1

    The cathode must not be connected to GND.

IP address setting

How to get IP and Channel Information

IP address = ConRTL Base Address (DIP switch) + GPCon2 Address(Rotary DIP switch).
For example, given Base Address = 192.168.0.100, the IP address becomes 192.168.0.102 when the rotary DIP switch value of GPCon2 is 2.
