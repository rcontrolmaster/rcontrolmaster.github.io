GData ConRTL

ConRTL provides Gigabit Ethernet connectivity and 24 V DC power to all modules in the GData system.

  • Gigabit Ethernet Switch
    Provides communication between the host PC and GData modules through the GDataBus backplane.
    The base address of ConRTL can be configured easily using the DIP switch.
  • Power
    +24 V DC power is distributed to the modules through the backplane.
    For convenience, two types of power connectors are provided. Both connectors are internally connected.
    • Molex Connector
      Part No. : 0039301020 (5569-2)
      Pin 1 : GND
      Pin 2 : +24 V DC
    • Barrel Connector
      Part No. : 54-00167
      ID : 2.1 mm
      OD : 5.5 mm
      Imax : 5 A

Base Address setting

ConRTL provides the base IP address for the modules in the GData system.
The user can change the IP address range by configuring the GPCon base address setting.

For the IP address 192.168.a.b:
a is set by bits 7..6, with a range of 0 ~ 3 and a default value of 0.
b is set by bits 5..0, with a DIP switch range of 0 ~ 32 and a default value of 100.
The actual value of b is calculated as (DIP switch bits 5..0) × 4 to support 2-bit expansion.
The default IP address is 192.168.0.100, corresponding to the DIP switch setting 00011001 (bit7..bit0).
After changing the DIP switch settings, power must be turned off and on again for the changes to take effect.
