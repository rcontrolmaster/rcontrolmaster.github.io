GData Controller

Overview

● The GData is ideal for applications that require multiple printhead drivers or additional functions such as LED encoders or general-purpose I/O.

● The modular design allows for easy customization and scalability based on specific application needs.

● Gigabit Ethernet connectivity for high-speed data transfer.

GData4 for 4 modules

● 4 modules can be installed in the GData4 main controller board.

GData7 for 7 modules

● 7 modules can be installed in the GData7 main controller board.

GData10 for 10 modules

● 10 modules can be installed in the GData10 main controller board.
